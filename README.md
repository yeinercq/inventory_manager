# Inventory Manager

This is a small project to manage a product inventory. You can save, delete and update providers, customers, products and save sales with different items (products) and something else.

## Technologies

- Rails 7.0.4
- Ruby 3.1.2
- Bootstrap 5.0
- Postgresql
- Turbo Drive
- CSS
